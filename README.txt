CurrentPaper

デスクトップに現在設定されている壁紙のファイルを確認します。

動作環境:
  Windows 10
  .NET Framework 4.8

ファイル:
  CurrentPaper.exe
    実行ファイル
  README.txt
    このファイル

使い方:
  1. CurrentPaper.exe を実行します。
     起動すると、[Wallpaper:]に現在設定されている壁紙ファイルの一覧が表示されます。
  2. Refresh をクリックすると、現在の情報に更新します。
     (切り替え間隔が設定されている場合等、変更後の情報に更新されます)
  3. [Wallpaper:]のいずれかを選択した状態で[Open]をクリックするかダブルクリックすると、
     当該ファイルが開きます。
  4. [Wallpaper:]のいずれかを選択した状態で[Explore]をクリックすると、
     エクスプローラでそのファイルが存在するディレクトリを開きます。
